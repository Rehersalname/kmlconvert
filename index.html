How to use

  1. Upload file
    Choose .xlsx, .xls, .csv, or .txt.
  2. Map columns
    Select your Latitude and Longitude columns. Optionally select Name and Description.
    Folder name (optional): only affects KML. If set, points are grouped inside a KML folder with that name. GPX (OsmAnd) ignores it.
  3. Choose output format
    KML works with Google Earth / Maps. GPX is a good choice for OsmAnd.
  4. Download
    Click Download. Rows with invalid coordinates are skipped.
TXT/CSV auto-detect
For .txt/.csv, the app tries to find the best latitude/longitude pair even if the line has extra mixed data.
